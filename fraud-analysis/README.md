Financial Fraud Detection Analysis

1. Objective
Detect fraud patterns in financial transactions using synthetic data.

2. Key Findings
- Highest fraud rate: TRANSFER transactions (8.2%)
- Peak fraud hours: 2:00 AM - 4:00 AM
- Fraud amount: 7.5x higher than legitimate

3. Technologies
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Faker

4. How to Run
python generate_fraud_data.py
python fraud_analysis.py
