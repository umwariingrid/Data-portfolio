African Online Mobile Usage Analysis

1. Objective
   Analyze mobile usage patterns across 10 African countries.
2. Key Findings
   - Highest data usage: Nigeria (425 MB avg)
   - Most active: Age 18-30 (3x more data)
   - Weekend usage: 40% more data, 30% fewer calls

3. Technologies
   - Python, Pandas, NumPy
   - Matplotlib, Seaborn
   - Faker

4. How to Run
python generate_mobile_data.py
python africa_mobile_analysis.py
